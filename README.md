# Axiom-Chinese-Simplified

这是一个汉化[Axiom](https://www.mcmod.cn/class/12270.html)的项目 由于Axiom的硬编码文本过多 所以选择使用[Vault Patcher](https://www.mcmod.cn/class/8765.html)这个mod来进行汉化

本项目使用的Vault Patcher版本为`1.4.6` 适用于`5.2.1-Fabric`版本的Axiom

## 汉化内容

· 汉化了除BlockState外基本上所有页面的内容 包括动画重建页面 由于原本的动画重建界面说的有些含糊不清 特地制作了新的class文件 添加了悬停注释

· 对Axiom本体中en_us语言文件中存在 但外部translation文件中不存在的翻译键值对进行了翻译和补充

## 使用方法

· 首先下载Vault Patcher 1.4.6和Axiom 5.2.1并运行

· 以zip格式下载本储存库并解压至版本根目录
